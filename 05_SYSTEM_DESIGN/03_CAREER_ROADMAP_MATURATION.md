# 🚀 Roadmap de Carreira e Estudos: ADS & Tech Transition

Este documento detalha a estratégia de transição de carreira do Marcel Trindade (Ilustrador Sênior) para a área de Tecnologia (Análise e Desenvolvimento de Sistemas), utilizando o ecossistema Nexus como laboratório prático.

## 🧠 1. Aprendizado Empírico (Teoria + Laboratório)
A meta é que o conhecimento acadêmico da Uniasselvi seja imediatamente aplicado na infraestrutura do Nexus.

*   **Ingestão de Conteúdo:** Tudo o que é oferecido pela faculdade (PDFs, Vídeos, E-books) deve ser indexado no AnythingLLM para consulta contextual.
*   **A IA como Tutor Sênior:** Em vez de respostas genéricas, a IA deve orientar o Marcel a implementar conceitos de aula (ex: Banco de Dados, Redes, Java) dentro dos projetos reais (IIVA, Buster, Nexus Management).

## 💼 2. Fábrica de "Cases" e Portfólio Vivo
O maior diferencial do Marcel é o histórico de 20 anos de trabalho profissional somado à construção manual de um ecossistema complexo (Nexus).

*   **Extração de Experiência:** Transformar os logs de manutenção do sistema (`nexus_logs`) em relatos técnicos para portfólio (ex: "Como estabilizei um servidor de mídia distribuído via Tailscale Mesh").
*   **Narrativa de Transição:** Unir as Soft Skills do Ilustrador (gestão de projetos, finalização, visão estética) com as Hard Skills de ADS (Python, SQL, Automação).

## 🎓 4. O Projeto Capstone Perpétuo: IIVA
O sistema **IIVA** (Idioma Independente Virtual Assistant) é a maior prova de conceito.
*   Ele resolve um problema real de negócio (gestão de aulas).
*   Ele integra Frontend (Dashboards), Backend (Scripts/APIs), Banco de Dados (Postgres) e IA (Agentes).
*   Cada funcionalidade nova na IIVA deve ser documentada como um marco acadêmico e profissional.
