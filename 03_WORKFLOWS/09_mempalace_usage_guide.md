# Guia de Uso do MemPalace para Agentes da Squad Nexus

## Visão Geral
O MemPalace é o sistema de memória compartilhada do ecossistema Nexus. Ele permite que os agentes acessem todo o conhecimento acumulado do ecossistema de forma semântica.

## Ferramentas Disponíveis

### Busca Semântica
- `mempalace_search "consulta"` - Busca conteúdo relevante no conhecimento do Nexus

### Status e Informações
- `mempalace_status` - Mostra o estado atual do palácio
- `mempalace_wake_up` - Carrega informações essenciais do ecossistema

### Gestão de Conteúdo
- `mempalace_add_drawer` - Adiciona novo conteúdo ao palácio
- `mempalace_list_wings` - Lista áreas temáticas
- `mempalace_list_rooms` - Lista tópicos dentro de uma área

## Casos de Uso para Agentes

### Consultor
- Use para entender o histórico de decisões do ecossistema
- Acesse o protocolo da Squad para entender fluxos de trabalho

### Arquiteto
- Consulte decisões arquiteturais anteriores
- Verifique conformidade com padrões estabelecidos

### Engenheiro
- Acesse especificações técnicas e procedimentos
- Consulte padrões de implementação

### Debugger
- Pesquise soluções de problemas anteriores
- Acesse logs de troubleshooting

### QA
- Consulte critérios de aceitação
- Acesse especificações de qualidade

## Melhores Práticas

1. Sempre use buscas semânticas em vez de busca textual
2. Prefira consultas específicas para resultados mais relevantes
3. Use o wake-up para carregar contexto essencial no início de sessões
4. Adicione novos conhecimentos ao palácio para manter informação atualizada
